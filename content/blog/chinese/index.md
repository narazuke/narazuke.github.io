## よく使う

| 単語 |  拼音  |         意味         |
| :--: | :----: | :------------------: |
| 所以 | suǒyǐ  |        …ので         |
|  和  |   hé   |         …と…         |
|  但  |  dàn   |         でも         |
| 想…  | xiǎng  |       want to        |
| 要…  |  yào   |     want to(強)      |
|  還  |  huán  |         まだ         |
| 可能 | kěnéng | たぶん、かもしれない |
| 比…  |   bǐ   |        …より         |

## 会話

|    単語    |       拼音       |       意味       |
| :--------: | :--------------: | :--------------: |
| 有什麼區別 | yòu shenme qubie | 違いはなんですか |
