## 概要
Windowsでctrlに割り当てられるショートカットは、Macではcontrolとcommandに割り振られている。
混乱しないために、ここで整理する。


## control [^]
| ショートカット | 動作           | 場所                           |
| -------------- | -------------- | ------------------------------ |
| `^` + `C`      | 中止           | ターミナル                     |
| `^` + `B`      | tmux操作       | ターミナル                     |
| `^` + `[`      | エスケープ     | vim                            |
| `^` + `@`      | ターミナル起動 | VScode |
| `^` + `¥`      | 画面分割       | VScode |
| `^` + `R`      | コマンド検索   | ターミナル                     | 

## command ⌘
| ショートカット | 動作                 | 場所                           |
| -------------- |:-------------------- | ------------------------------ |
| `⌘` + `C`      | コピー               | どこでも                       |
| `⌘` + `P`      | ファイルを開く       | VScode |
| `⌘` + `O`      | ファイルパレット表示 | Obsidian                   |
| `⌘` + `T`      | 今日のノートを表示   |  Obsidian                               |
